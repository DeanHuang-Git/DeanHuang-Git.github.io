Step 2: Create a function that has the length of wire as input and return the number of unique combinations of integer sided right triangle.

 1) Since c = m^2 + n^2, you only have to check m for up to (m/2)^0.5 times. In addition, m is greater than 1 because m must be greater than n (n must be         greater than 0).

 2) Loop through the m values ranging from 2 to (length of wire/2)^0.5 and loop through n ranging from values smaller than m.
